Código-fonte e Pré-formatação

Considere a citação de trechos de código ou comandos, como: document.getElementById('teste') fica melhor visível com fontes monoespaçadas. Desta forma: document.getElementById('teste').

Trecho de código Python envelopado apenas com tag CODE:

num = int(input("Digite um número:")) if num % 2 == 0: print(f"O número {num} é par.") else print(f"O número {num} é ímpar.") print("Fim do Programa")

Mesmo trecho envelopado com as tags PRE e CODE:

        
    num = int(input("Digite um número:"))
    if num % 2 == 0:
        print(f"O número {num} é par.")
    else
        print(f"O número {num} é ímpar.")
    print("Fim do Programa")
